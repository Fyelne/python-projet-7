# Gesture Controller

## Installation

Install Python 3.11

### Linux

```sh
python3.11 -m venv venv  
source venv/bin/activate
pip install -r requirements.txt
```

## Get Started

### Start Minecraft Server

#### Linux

```sh
./Minecraft/start.sh
```

#### Windows

```shell
cd .\Minecraft\
start.bat
```
